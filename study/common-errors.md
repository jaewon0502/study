# Common Errors

## IndentationError: expected an indented block after 'if'
Cause:
- Missing indentation after an if statement.
- Mixing tabs and spaces.

Solution:
- Indent the next line using 4 spaces.
- Use a consistent indentation style.

# 자주 발생하는 오류

## IndentationError: expected an indented block after 'if'
원인:
- if 다음 줄에 들여쓰기가 없음
- 탭과 스페이스를 섞어서 사용함

해결:
- if 아래 코드를 스페이스 4칸으로 들여쓰기
- 에디터에서 들여쓰기 형식을 통일
